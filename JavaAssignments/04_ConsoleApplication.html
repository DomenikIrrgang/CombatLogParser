04_ConsoleApplication



This assignment is about creating a console application that can execute commands. You should be able to add or remove Commands dynamically from the ConsoleApplication.

  • Implement a class "ConsoleApplication".
    • Implement the function: public void run();
      • This function should run as long as the variable "runnning" is set to true.
      • For each run of the loop in this function get an input from the command shell.
    • Implement the function: public void close();
      • When this function is called the run() function should terminate.
  • Implement a abstract class "Command".
    • Implement the constructor with the following arguments: ConsoleApplication consoleApplication, String ... args
    • Declare the function: public boolean execute(ConsoleApplication consoleApplication, String ... args);
    • Declare the function: public boolean undo(ConsoleApplication consoleApplication, String ... args);
    • Implement the function: public String getCommandString(); (e.g. C:\>cd Users) cd = CommandString, args= Users
  • Add to class "ConsoleApplication"
    • Implement the function: public void addCommand(Command command);
    • Implement the function: public void removeCommmand(Command command);
    • Implement the function: private Command createCommand(String commandName, String ... args); This function should create a new Instance of a Command and return it.
    • Implement the function: public boolean executeCommand(Command command); This function should execute the given Command.
    • Implement the function: public boolean undoLastCommand(); Undo's the last Command that has been executed.
